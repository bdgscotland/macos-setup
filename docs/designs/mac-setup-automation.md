## Vision

### 10x Check
Not just "bootstrap a Mac" but a developer workstation operating system. The repo defines what's installed AND how it behaves. Machine-to-machine diff command. Auto-generated docs site. Weekly drift reports. The canonical "how to set up a Mac for development in 2026" reference.

### Platonic Ideal
Clone the repo, run one command, 20 minutes later have a fully configured dev environment. Ghostty opens with a beautiful Starship prompt. Git is configured with signing. Every language runtime is ready. A motd shows "This machine matches repo spec at commit abc123. 0 drift items." Push the machine profile back and the docs site auto-updates.
